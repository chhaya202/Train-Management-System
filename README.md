# Train Management System

A desktop-based Train Management System developed using Python, Tkinter, and SQLite.

## Features

* User Registration and Login
* Add Train Details
* View Available Trains
* Book Train Tickets
* Database Integration using SQLite

## Technologies Used

* Python
* Tkinter
* SQLite

## Project Description

This project is designed to manage train booking operations through a user-friendly graphical interface. Users can register, log in, view trains, and book tickets. The application stores data using SQLite database.
